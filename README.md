## Project 3 - HTML & CSS Update

### Description
This project is an updated version of my Edward Snowden HTML foundation page, now enhanced with a full CSS stylesheet. The site explores the debate around whether Snowden was a whistleblower or a traitor, and includes a detailed timeline of key events from 2012 to the present. The redesign features a dark surveillance-themed aesthetic with red accents, animated effects, and a more polished layout throughout.

### What I Learned
- How to style a multi-page website using a single shared CSS file
- How to use CSS variables to maintain a consistent color theme across pages
- How to create responsive layouts using CSS Grid and Flexbox
- How to add visual effects like glitch animations, scanlines, and hover transitions using pure CSS
- How to build a vertical timeline layout using CSS positioning
- How to use Google Fonts to improve typography across a webpage

### Challenges
- Getting the two-column debate layout to align correctly using CSS Grid
- Designing a vertical timeline with alternating left and right cards
- Fine-tuning the glitch animation on the heading to look natural without being distracting
